# Commands

Give permission ``lawsCity.createMayor`` to your role in ``~/LabAPI/configs/permissions.yml``

### RemoteAdmin

- ``mayor`` → id mayor.
- ``mayor <id>`` → create/delete mayor.

### ClientCommand

- ``.laws add <text>`` → add law (only for mayor).
- ``.laws clear`` → delete laws (only for mayor).
- ``.laws list`` → laws list.

# Installation

``lawsCity.dll`` move to ``~\LabAPI\plugins\global``
